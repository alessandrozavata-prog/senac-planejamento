# Planejamento Pedagógico SENAC

Gerador automático de planejamento pedagógico para cursos do SENAC SP, usando IA (Google Gemini).

## Acesso

🔗 **[Abrir o aplicativo](https://alessandrozavata-prog.github.io/senac-planejamento)**

## Funcionalidades

- Cole o Plano de Curso (P.O.) de qualquer curso SENAC
- Arraste arquivo .txt, .pdf ou .docx do P.O.
- Selecione a UC desejada (detectada automaticamente)
- Configure carga horária, duração das aulas e perfil da turma
- Gera planejamento completo com 10 etapas pedagógicas:
  - Resumo da UC
  - Matriz pedagógica
  - Análise de complexidade
  - Distribuição da carga horária
  - Progressão pedagógica
  - Situações de aprendizagem
  - Metodologias ativas (Banco SENAC SP)
  - Planos de aula detalhados
  - Auditoria pedagógica
  - Relatórios de cobertura
- Exporta em .txt e .docx no padrão SENAC

## Configuração

1. Acesse o aplicativo pelo link acima
2. Vá em **API Key Gemini** na barra lateral
3. Obtenha sua chave gratuita em [aistudio.google.com](https://aistudio.google.com/apikey)
4. Cole a chave e clique em Salvar

## Tecnologias

- HTML + CSS + JavaScript (sem framework)
- Google Gemini API (gratuita)
- PDF.js para leitura de PDFs
- Mammoth.js para leitura de Word
- docx.js para geração de Word
