Chave de API Gemini

Use a API gratuita do Google Gemini — sem cartão, sem cobrança.

Google Gemini API — 100% Gratuita
Sem custo! O Google oferece a API do Gemini gratuitamente. Basta uma conta Google.
Como obter sua chave gratuita (2 minutos):

1. Acesse aistudio.google.com
2. Faça login com sua conta Google
3. Clique em "Get API Key" → "Create API Key"
4. Copie a chave gerada (começa com AIza...)
5. Cole abaixo e clique em Salvar
Salva no navegador — disponível mesmo após fechar e reabrir o arquivo.
Como usar
  1. Configure sua chave gratuita do Gemini nesta tela
  2. Na Etapa 1, cole o P.O. de qualquer curso SENAC
  3. Na Etapa 2, selecione a UC desejada
  4. Na Etapa 3, informe carga horária e perfil da turma
  5. Na Etapa 4, clique em gerar e aguarde o planejamento completo
  6. Repita a partir da Etapa 2 para outras UCs do mesmo curso
